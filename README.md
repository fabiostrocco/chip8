# Build

## Windows

Open builds\vs\chip8.sln and build

## Linux

Run

> sudo apt install cmake libsdl2-dev g++
> cd builds/cmake
> cmake .
> make

# Assembler

Use nasm hello_world.nasm to create a chip8 file (taken from https://github.com/mfurga/chip8)

ROMS: https://github.com/kripod/chip8-roms
